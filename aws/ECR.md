# Amazon Elastic Container Registry (ECR)

## Overview
Amazon ECR is used to store and manage Docker images for the Flask application.

## Workflow
1. Build Docker image locally
2. Authenticate Docker with ECR
3. Push image to repository
4. Pull image on EC2 instance for deployment

## Benefits
- Secure container storage
- Versioned image management
- Seamless integration with EC2 and ECS
